# nuxt-cors-sandbox

Requires local CA that creating by [mkcert](https://github.com/FiloSottile/mkcert) etc.

1. Run api
```sh
$ make run-api
```
2. Run gateway
```sh
$ make run-gateway
```
3. Run nuxt
```sh
$ make run-nuxt
```
4. Access to localhost website
```sh
$ open https://localhost:8080
or
$ open http://localhost:3000
```
